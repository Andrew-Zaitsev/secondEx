# FSD TOXIN hotel booking site

## See demo:

  <a href='https://andrew-zaitsev.github.io/secondEx/'>Demo</a>


## Clone project
```
  $ git clone https://github.com/Andrew-Zaitsev/secondEx.git
```

## Install project
```
  $ npm i
```

## Development mode
```
  $ npm run dev
```

## Build project
```
  $ npm run build
```

## Publish project
```
  $ npm run deploy
```
